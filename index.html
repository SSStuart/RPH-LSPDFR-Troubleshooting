Rage Plugin Hook

Version du jeu incompatible

Erreur

[Voir la résolution]

RPH n'est pas compatible avec la version de GTA 5 installé.


Résolution

  • Si utilisé pour jouer à LSPDFR : réinstallez LSPDFR (manual install) en remplaçant les fichiers

    Allez sur lcpdfr.com.
    Cliquez sur "Download this file" puis "Agree & Download".
    Cliquez sur le bouton "Download" correspondant à la version Manual_Install.
    Une fois le fichier téléchargé, extrayez son contenu dans la racine du jeu.

    Comment installer LSPDFR Manual Install ?

    Ouvrez l'archive .zip d'un côté, puis ouvrez la racine du jeu de l'autre (dans l'explorateur), puis sélectionner tous les fichiers (Ctrl+A) de l'archive et cliquez-glissez dans la racine. Remplacez les fichiers déjà existants si demandé.

  • Attendre une mise à jour de RPH sur le site ( à priori plus maintenu).
    (Ou éventuellement récupérer le RAGEPluginHook.exe de LSPDFR)

Permissions insuffisantes ou mauvais antivirus

Erreur

[Voir la résolution]

  • RPH n'est pas installé au bon endroit.

  • L'antivirus bloque des fichiers dans la racine du jeu.

  • RPH n'a pas les permissions suffisantes.

(Image de kiwiw#7986)

Résolution

  • Option 1

    Vérifiez que RAGEPluginHook.exe est bien placé dans la racine d'origine de votre jeu (et non pas une copie).

  • Option 2

    Dans les paramètres de votre antivirus, ajouter une exception au dossier du jeu.
    Avast Sécurité Windows McAfee (Désinstallation) Vous avez un autre antivirus ? Précisez-le sur le #need-help

  • Option 3

    Lancez GTA 5, puis une fois en jeu (mode histoire, prologue fini au minimum) lancez RPH.

  • Option 4

    Faites un clique-droit sur le dossier du jeu > "Propriétés" > onglet "Sécurité" > "Modifier..." > pour chaque utilisateurs cocher la case "Contrôle total" sous "Autoriser", puis "Appliquer".

  • Option 5

    Si vous avez (Nvidia) GeForce Experience : désactivez la "Superposition en jeu" dans les paramètres généraux de l'application GeForce Experience.

  • Option 6

    RPH peut créer un fichier caché dans la racine qui peut être à l'origine de l'erreur.
    Pour le supprimer : lancez un cmd en tant qu'administrateur (Windows + R, entrez cmd puis appuyez sur Ctrl + Maj + Entrée) puis entrez la commande suivante : del [Chemin de la racine]\rphhs.

RPH/GTA 5 a crashé

Erreur

[Voir la résolution]

RPH a fait crasher le jeu.



Résolution

Lorsque le message d'erreur de RPH "The game has crashed!" apparaît, cliquez sur "Generate crash report" (voir 2ème image).
Une fois le rapport généré, copiez-le et envoyez-le dans le #need-help.

Vous pouvez également essayé les résolutions de l'erreur "Permissions insuffisantes ou mauvais antivirus".

Changer le raccourci clavier de la console RPH

Erreur

[Voir la résolution]

Le changement du raccourci clavier de la console n'est pas enregistré/appliqué.

Résolution

Lancez GTA 5 (via RPH), puis une fois en jeu, ouvrez la console (F4 par défaut) et écrivez ConsoleKey F5 (en remplaçant 'F5' par la touche désirée).

LSPDFR

Plugin arrêté car il freeze le jeu

Erreur

[Voir la résolution]

RPH arrête un plugin au chargement du jeu parce qu'il bloque ce dernier.

Résolution

  • Option 1

    Allez dans les paramètres de RPH dans l'onglet "General".
    Changez la valeur de "Plugin timeout threshold" à 15000 (15 secondes).
    (Dans tous les cas, pensez à garder la fenêtre du jeu au premier plan pendant le chargement)

  • Option 2

    Allez dans les paramètres de RPH dans l'onglet "Plugins".
    Sélectionnez "Don't load any plugins on startup".
    Une fois en jeu, ouvrez la console RPH (F4 par défaut), puis écrivez ReloadAllPlugins.

LSPDFR a crashé

Erreur

[Voir la résolution]

LSPDFR crash (mais pas le jeu).

Résolution

Ce type d'erreur peut avoir plusieurs causes (en général lié à un plugin qui ne fonctionne pas).

Ouvrez la console RPH (F4 par défaut), agrandissez-la (en "tirant" le bas de la console vers le bas) puis faites une capture d'écran.
Envoyez la capture d'écran dans le #need-help.
