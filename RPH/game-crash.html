Résolution

Lorsque le message d'erreur de RPH "The game has crashed!" apparaît, cliquez sur "Generate crash report" (voir 2ème image).
Une fois le rapport généré, copiez-le et envoyez-le dans le #need-help.

Vous pouvez également essayé les résolutions de l'erreur "Permissions insuffisantes ou mauvais antivirus".
