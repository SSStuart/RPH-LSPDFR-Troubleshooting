Résolution de problèmes communs pour Rage Plugin Hook & LSPDFR

Plugin arrêté car il freeze le jeu

Erreur

[Voir la résolution]

RPH arrête un plugin au chargement du jeu parce qu'il bloque ce dernier.

Résolution

  • Option

    Allez dans les paramètres de RPH dans l'onglet "General".
    Changez la valeur de "Plugin timeout threshold" à 15000 (15 secondes).
    (Dans tous les cas, pensez à garder la fenêtre du jeu au premier plan pendant le chargement)

  • Option

    Allez dans les paramètres de RPH dans l'onglet "Plugins".
    Sélectionnez "Don't load any plugins on startup".
    Une fois en jeu, ouvrez la console RPH (F4 par défaut), puis écrivez ReloadAllPlugins.
